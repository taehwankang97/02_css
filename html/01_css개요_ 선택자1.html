CSS(cascading Style Sheets)

    마크업 언어(HTML)가 실제 화면에 표시되는 방법을 
    기술하는 언어
    
    하나의 요소에 CSS 코드를 겹쳐서 원하는 스타일을 만드러냄

    W3C에서 CSS를 웹 표준으로 지정

CSS 선택자 (CSS selector)


CSS 코드를 작성하는 방법

  1. head태그에 스타일 태그를 작성한후 내부에 CSS코드를 작성
  2. CSS가 적용되기 원하는 요소에 직접 CSS를 적용하는 방법 (inline-style)
  3. 외부에 .CSS파일을 생선하여 link 태그를 이용해 연결하는 방법

id(#) 선택자

    HTML 문서에서 id속성값이
     일치하는 요소를 선택하는 선택자

     [작성법]

    
      #id속성값 {CSS코드;}

태그 선택자

    html 문서 내에서 
    태그명이 일치하는 모든 요소를 찾아
    선택하는 선택자
    [작성법]
    
      태그명{CSS 코드;}

class(.) 선택자

      HTML 문서 내에서 
      Class 속겅 값이 일치하는 
      모든 요소를 선택하는 선택자
      * 하나의 요소가 여러 클래스를 가질수 있다!

    [작성법]
      
        .class속성값 {CSS 코드;}
      

      
        요소의 CSS적용을 위해 사용하는 선택자
        보통 Class 선택자를 가장 많이 이용 

        (id 선택자는 주로 JS용도로 많이 사용)

모든 요소(*, asterisk) 선택자

        단독 사용 : 
          문서에 존재하는 모든 요소 선택

        
          *{css 코드;}
        

        다른 선택자와 혼용:  
          선택된 요소의 모든 자식/후손 선택

        
          A선택자 *{CSS 코드;}

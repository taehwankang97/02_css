CSS 선택자 우선순위

    CSS는 기본적으로 style 태그는 또는 .CSS파일에 작성된 순서
    대로 해석이 진행되지만 

    같은 요소에 여러 CSS속성이 설정 되는 경우
    위치나, 선택자, 등에 따라서 우선순위가 결정 된다.

    
      [꼭 알고 있어야 하는것]
    

    1)동일한 우선순위로 css속성이 설정된 경우
      ->제일 마지막에 작성된 css 속성이 반영된다

    2) 여러 선택자를 이용해서 같은 요소를 선택하더라도
        지정된 css속성이 다른 경우 모두 반영된다

    


    1.순위 : !important(요소에 직접 작성)

    2.순위 : inline-style (요소에 직접 작성)

    3.순위 : 아이디 선택자 (#아이디 속성값)

    4.순위 : 클래스 선택자 (.class명)

    5.순위 : tag 선택자(태그명)
